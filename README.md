# 🤖 VibeBot - Simple Chat Bot

A friendly, interactive chat bot written in Python that loves to have conversations!

## Features

- 👋 Greets users with random friendly messages
- 😊 Responds to "how are you" questions
- 🎭 Tells jokes on request
- ⏰ Tells the current time
- 💬 Handles general conversation with engaging responses

## Getting Started

### Prerequisites

- Python 3.6 or higher

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd chatbot
```

2. Run the bot:
```bash
python3 bot.py
```

### Usage

Start chatting by typing messages and pressing Enter. The bot will respond to your input.

Type `quit`, `exit`, or `bye` to end the conversation.

### Example Interactions

```
You: Hi!
VibeBot: Hey there! 👋 I'm VibeBot, ready to chat!

You: Tell me a joke
VibeBot: Why do programmers prefer dark mode? Because light attracts bugs! 🐛

You: What time is it?
VibeBot: The current time is 14:30:45 ⏰

You: quit
VibeBot: Goodbye! Come back and chat anytime! 👋
```
